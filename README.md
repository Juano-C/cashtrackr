- Proyecto CASHTRACKR

Descripcion:
    Pagina web para poder administrar gastos de algun presupuesto, vacaciones, casa nueva, ropa, etc. 

Caracteristicas:
    Podes crear una cuenta, con su seguridad de contraseña y verificacion. 
    Tiene uso de un token de seguridad y verificacion de email unico.
    Analisis de gastos en cada presupuesto.
    Posibilidad de configurar perfil, cambiar contraseña, cambiar nombre, cambiar email.

backend:
    express

frontend:
    nextjs
    serveraction
    servercomponent

base de dato:
    sequelize
    postgresql

diseño:
    tailwindcss

testing unitario/integracion
